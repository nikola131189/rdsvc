# rd
Это клиент серверное  приложение для удаленного управления компьютером. Клиенты через сервер обмениваются зашифрованными пакетами, в которых передаются экран, файлы, комнды из терминала, благодаря чему можно обойти нат. Есть версии для Windows и Linux.
